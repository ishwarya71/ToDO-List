# 📝 To-Do List Web App

A simple and interactive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. This project allows users to manage tasks efficiently with options to add, edit, delete, mark tasks as complete, and filter based on task status and priority.

---

## 🚀 Features

- ✅ Add new tasks with priority and due date
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✔️ Mark tasks as complete/incomplete
- 🔍 Filter tasks by status (All / Active / Completed)
- 🗃️ Tasks saved in browser using **localStorage**
- 🌄 Stylish background image for better UI

---

## 💻 Technologies Used

- **HTML5** – structure of the web page  
- **CSS3** – styling and layout  
- **JavaScript (ES6)** – dynamic behavior and logic  
- **localStorage** – for saving tasks persistently in the browser
